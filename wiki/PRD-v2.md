# PRD v2

이 문서는 위키용 요약본이다.

정본 PRD는 [`docs/PRD-v2.md`](../docs/PRD-v2.md)다.

## 요약

- BeanTracker는 홈카페 사용자의 원두 소비와 원두별 누적 잔 수, 월별 커피 루틴을 추적하는 개인용 앱이다.
- 핵심 루프는 원두 등록 -> 추출 기록 -> 이 원두로 N잔째 확인 -> 원두 소진 시 총 잔 수 확인이다.
- 상세 기능 요구사항과 계산 규칙은 정본 PRD를 따른다.

## 제외 범위

- 레시피 저장
- 브루잉 방식 기록
- 다중 잔 추출
- 목표 프로그레스 바
- 소셜 기능

# PRD v2

## 제품 정의

BeanTracker는 홈카페 사용자가 원두 소비를 최소 입력으로 기록하고, 월별 절약액을 명확하게 확인하게 해주는 개인용 대시보드 앱이다.

## 해결하려는 문제

- 집에서 커피를 마실 때 실제 절약 금액이 체감되지 않는다.
- 원두 소비량을 관리하고 싶지만 기록이 번거로우면 사용이 끊긴다.

## 핵심 가치

- 이번 달 절약액을 가장 먼저 보여준다.
- 기록 직후 절약액 증가 애니메이션으로 보상을 준다.
- 최소 입력만으로 추출을 기록하게 한다.

## 주요 규칙

- 1회 추출은 1잔이다.
- 기본 원두는 최근 사용한 활성 원두다.
- 기본 사용량은 마지막 사용량이다.
- 활성 원두는 여러 개 허용한다.
- 소진 예상은 자동 판단하고, 최종 소진 상태는 사용자가 정한다.
- 원두와 추출 기록은 수정하지 않고 삭제만 허용한다.

## 1.0 핵심 기능

- 온보딩에서 기준 카페 가격 입력
- 원두 등록 및 활성/소진 관리
- 빠른 추출 기록
- 대시보드에서 이번 달 절약액 표시
- 월별 차트와 상세 통계
- 소형 위젯에서 빠른 기록 진입

## 제외 범위

- 레시피 저장
- 브루잉 방식 기록
- 다중 잔 추출
- 목표 프로그레스 바
- 소셜 기능
